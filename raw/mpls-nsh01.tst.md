=== "Configuration"
    
    **r1:**
    ```
    hostname r1
    buggy
    !
    logging file debug ../binTmp/zzz46r1-log.run
    !
    vrf definition tester
     exit
    !
    vrf definition v1
     rd 1:1
     exit
    !
    interface ethernet1
     vrf forwarding v1
     ipv4 address 1.1.1.1 255.255.255.0
     ipv6 address 1111::1 ffff::
     no shutdown
     no log-link-change
     exit
    !

    **r2:**
    ```
    hostname r2
    buggy
    !
    logging file debug ../binTmp/zzz46r2-log.run
    !

     exit
    !
    interface ethernet1
     nsh enable
     nsh xconnect 2 255
     no shutdown
     no log-link-change
     exit
    !
