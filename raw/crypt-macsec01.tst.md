=== "Configuration"
    
    **r1:**
    ```
    hostname r1
    buggy
    !
    logging file debug ../binTmp/zzz75r1-log.run
    !
    crypto ipsec ips
     group 02
     cipher aes256cbc
     hash sha1
     key $v10$dGVzdGVy
     exit
    !
    vrf definition tester
     exit
    !
    vrf definition v1
     rd 1:1
     exit
    !
    interface ethernet1
     macsec ips
     vrf forwarding v1
     ipv4 address 1.1.1.1 255.255.255.0
     ipv6 address 1234::1 ffff::
     no shutdown
     no log-link-change
     exit
    !
    !

     security protocol telnet
     no exec authorization
     no login authentication
     vrf tester
     exit

    !
    end
    ```
