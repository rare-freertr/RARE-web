=== "Configuration"
    
    **r1:**
    ```
    hostname r1
    buggy
    !
    logging file debug ../binTmp/zzz12r1-log.run
    !
    vrf definition tester
     exit
    !
    vrf definition v1
     rd 1:1
     exit
    !
    interface ethernet1
     vrf forwarding v1
     ipv4 address 1.1.1.1 255.255.255.252
     ipv6 address 1234:1::1 ffff:ffff::
     no shutdown
     no log-link-change
     exit

    **r2:**
    ```
    hostname r2
    buggy
    !
    logging file debug ../binTmp/zzz12r2-log.run

     sequence 10 permit all 1.1.1.4 255.255.255.252 all 1.1.1.0 255.255.255.252 all

    !

     sequence 10 permit all 1234:2:: ffff:ffff:: all 1234:1:: ffff:ffff:: all
