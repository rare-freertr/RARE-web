=== "Configuration"
    
    **r1:**
    ```
    hostname r1
    buggy
    !
    logging file debug ../binTmp/zzz47r1-log.run
    !
    bundle 1
     no ethernet
     exit
    !
    vrf definition tester
     exit
    !
    vrf definition v1
     rd 1:1
     exit
    !
    interface bundle1
     vrf forwarding v1
     ipv4 address 1.1.1.1 255.255.255.0
     ipv6 address 1234::1 ffff::
     no shutdown
     no log-link-change
     exit
    !
    interface serial1
     encapsulation hdlc
     bundle-group 1
     no shutdown
     no log-link-change
     exit
    !
    interface serial2
     encapsulation hdlc
     bundle-group 1
     no shutdown
     no log-link-change
     exit
    !
    !
    !

    server telnet tester
     security protocol telnet
     no exec authorization
     no login authentication

    end
    ```
